# Custom Nodes — Module 01: LLM Prompt Enhancer

Install via ComfyUI Manager → Install Missing Custom Nodes.

| Node Pack | Repo | Key Nodes |
|-----------|------|-----------|
| ComfyUI-WJNodes | [807502278/ComfyUI-WJNodes](https://github.com/807502278/ComfyUI-WJNodes) | `RegexExtract`, utility nodes |
| comfyui-ollama | [stavsap/comfyui-ollama](https://github.com/stavsap/comfyui-ollama) | `OllamaConnectivityV2`, `OllamaGenerateV2` |

**Prerequisite:** Ollama must be installed and running separately before the workflow executes. The comfyui-ollama node connects to Ollama's local API at `http://127.0.0.1:11434`.

Install Ollama: https://ollama.com/download
