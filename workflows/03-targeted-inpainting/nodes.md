# Custom Nodes — Module 03: Targeted Inpainting

Install via ComfyUI Manager → Install Missing Custom Nodes.

| Node | Purpose |
|------|---------|
| `InpaintCropImproved` | Crops to mask region with context padding |
| `InpaintStitchImproved` | Composites inpainted region back into original seamlessly |
| `TextEncodeQwenImageEditPlus` | Image-aware conditioning for Qwen Image Edit |
| `DifferentialDiffusion` | Controls blend boundary at inpaint edges |
| `ModelSamplingAuraFlow` | Advanced sampling for Qwen-family models |

These nodes are available via ComfyUI Manager. Search by node name to find the correct package.
