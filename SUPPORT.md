# Support

## Support Level

**Maintained** — This project is actively maintained by NVIDIA.

## How to Get Help

- **Bug reports:** Open a [GitHub Issue](https://github.com/NVIDIA/Creative-GenAI-Workflows/issues) using the Bug Report template.
- **Feature requests:** Open a [GitHub Issue](https://github.com/NVIDIA/Creative-GenAI-Workflows/issues) using the Feature Request template.
- **General questions:** Open a [GitHub Discussion](https://github.com/NVIDIA/Creative-GenAI-Workflows/discussions).

## Security Issues

Do **not** file public issues for security vulnerabilities. See [SECURITY.md](SECURITY.md) for the responsible disclosure process.

## Response Expectations

We aim to triage new issues within **5 business days**. Response times may vary based on complexity and team availability.
