# Governance

## Maintainers

This project is maintained by NVIDIA. See [MAINTAINERS.md](MAINTAINERS.md) for the current list of maintainers.

## Decision Making

Project decisions (roadmap, breaking changes, major new features) are made by the maintainers listed in MAINTAINERS.md. For significant changes, open a GitHub Issue or Discussion to allow community input before implementation.

## Contributions

All contributions are subject to review and approval by a maintainer. See [CONTRIBUTING.md](CONTRIBUTING.md) for the contribution process, including the Developer Certificate of Origin (DCO) sign-off requirement.

## Code of Conduct

All participants are expected to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
